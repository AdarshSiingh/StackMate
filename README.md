# StackMate

StackMate is a platform that helps developers find teammates, share project ideas, and collaborate on real-world projects.

## 🚀 Features

- Find developers interested in similar projects
- Share and discover project ideas
- Collaborate with other developers
- Build real-world projects together

## 🛠 Tech Stack

Frontend:
- React
- Tailwind CSS
- Vite
- React Router

Backend:
- Node.js
- Express

## 📂 Project Structure

```
StackMate
│
├── client        # React frontend
├── Backend       # Express backend
└── README.md
```

## ⚙️ Setup

Clone the repo:

```
git clone https://github.com/AdarshSiingh/StackMate.git
```

Install dependencies:

Frontend

```
cd client
npm install
npm run dev
```

Backend

```
cd Backend
npm install
node index.js
```
